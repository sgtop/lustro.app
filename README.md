# LUSTRO Field System

App 100% offline-first, mobile-first, single HTML.

## Como usar
1. Abre `index.html` no Chrome/Safari (recomendado no telemóvel)
2. "Adicionar ao ecrã principal" → usa como PWA
3. Todos os dados ficam guardados em localStorage (no teu dispositivo)

## Funcionalidades
- Dashboard + MRR
- Clientes (CRM com pipeline + estado Ativo/Pendente/Inativo)
- Agenda automática com displacement buffer
- Preços (B2B / Moradias / Pós-Obra)
- Documentos PDF (Proposta, Contrato, Tabela B2B, Moradias, Pós-Obra, Prova)
- Assinaturas duais (Cliente + Prestador)
- Pagamentos: GoCardless, MBWay, IBAN
- Partilha nativa de PDF (WhatsApp / Email / outras apps)
- Backup JSON (exportar / importar)

## Requisitos
- Browser moderno com Web Share API (Chrome Android, Safari iOS)
- Nenhum servidor necessário
